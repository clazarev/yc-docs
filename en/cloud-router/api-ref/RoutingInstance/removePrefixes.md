# Cloud Router API, REST: RoutingInstance.RemovePrefixes

Removes specified prefixes from a RoutingInstance resource.
Method starts an asynchronous operation that can be cancelled while it is in progress.

## HTTP request

```
POST https://cloudrouter.{{ api-host }}/cloudrouter/v1/routingInstances/{routingInstanceId}:removePrefixes
```

## Path parameters

#|
||Field | Description ||
|| routingInstanceId | **string**

Required field. ID of the RoutingInstance resource. ||
|#
