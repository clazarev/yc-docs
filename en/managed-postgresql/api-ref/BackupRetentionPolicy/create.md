# Managed Service for PostgreSQL API, REST: BackupRetentionPolicy.Create

Add a new retention policy.

## HTTP request

```
POST https://{{ api-host-mdb }}/managed-postgresql/v1/clusters/{clusterId}/retention_policies
```

#|
