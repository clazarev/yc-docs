# Users, REST: User.List

List users in a specific folder.

## HTTP request

```
GET https://rest-assistant.{{ api-host }}/users/v1/users
```

## Query parameters {#yandex.cloud.ai.assistants.v1.users.ListUsersRequest}

|| pageSize | **string** (int64)
